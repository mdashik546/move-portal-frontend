📽️ Movie Vault

A premium subscription-based movie streaming platform built with Next.js, Node.js, and Stripe.

🔗 Live Demo
Frontend:  https://movievault-project.vercel.app/

Backend API: https://movievault-api.vercel.app/


📝 Overview

Movie Vault is a full-stack movie streaming portal where users can browse movies and unlock premium content through a secure 30-day subscription system.

The platform includes secure payments, real-time subscription updates, protected routes, and role-based dashboards for both admins and users.

✨ Features
💳 Secure 30-day subscription system using Stripe
⚡ Real-time premium access with Stripe Webhooks
👑 Admin dashboard for revenue and analytics
🎬 User dashboard for subscriptions and payment history
🔐 Role-based protected routes
🌙 Responsive cinematic dark UI
📱 Mobile-friendly design
